# Bulk Vegetable/Fruit Ordering App

A responsive platform for buyers and admins to manage bulk orders.

## Features
- Browse Products
- Place Orders
- Track Orders
- Admin Dashboard
- Update Order Status
- Manage Products

## Tech Stack
- Next.js 14
- TailwindCSS
- PostgreSQL (Neon.tech)
- Prisma ORM

## Setup
1. Clone repo
2. Setup `.env` file
3. Run `npm install`
4. Migrate DB: `npx prisma db push`
5. Run app: `npm run dev`
